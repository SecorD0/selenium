SlowLoadableComponentT ClassWebDriver
A LoadableComponentT which might not have finished loading when Load() returns. After a call to Load(), the IsLoaded property should continue to return false until the component has fully loaded. Use the HandleErrors() method to check for error conditions which caused the Load() to fail. 

            new SlowHypotheticalComponent().Load();
