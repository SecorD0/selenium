Key.chord( ...var_args ) â†’ string

Simulate pressing many keys at once in a "chord". Takes a sequence of keys or strings, appends each of the values to a string, adds the chord termination key (Key.NULL) and returns the resulting string.

Note: when the low-level webdriver key handlers see Keys.NULL, active modifier keys (CTRL/ALT/SHIFT/etc) release via a keyup event.

Parameters
var_args...string

The key sequence to concatenate.

Returns
string

The null-terminated key sequence.
