Class TouchSequence

Exported By
  • selenium-webdriver
Alias for
  • selenium-webdriver/lib/actions.TouchSequence

Class for defining sequences of user touch interactions. Each sequence will not be executed until #perform is called.

This class should not be instantiated directly. Instead, obtain an instance using WebDriver.touchActions().

Sample usage:

 driver.touchActions().
     tapAndHold({x: 0, y: 0}).
     move({x: 3, y: 4}).
     release({x: 10, y: 10}).
     perform();

new TouchSequence( driver )

Parameters
driverWebDriver

The driver that should be used to perform this action sequence.
