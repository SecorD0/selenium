Class Profile

Exported By
  • selenium-webdriver/firefox
Alias for

Models a Firefox profile directory for use with the FirefoxDriver. The Profile directory uses an in-memory model until #writeToDisk or #encode is called.

new Profile( opt_dir )

Parameters
opt_dir(string|undefined)

Path to an existing Firefox profile directory to use a template for this profile. If not specified, a blank profile will be used.
